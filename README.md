## 项目说明

- 本项目完成了一个Android端的时钟APP，实现了数码表盘和机械表盘，秒表和计时器等功能。

### 界面逻辑：

- 导航栏：时间、秒表、计时器

- 时间页面：可以左右滑动切换数码表盘和机械表盘，其中机械表盘动画由组员自己绘制。

- 秒表页面：上半部以电子表盘的形式显示当前记录的秒数，下半部设有计次，开始/复位/停止按钮。

- 计时器页面：上半部供用户选择计时时长，且在开始计时后切换为倒计时页面。下半部分是开始，重置按钮。

### 特色功能：

- 实现了黑夜模式，保护眼睛~
